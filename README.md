# ts-node-template
Template for TypeScript web projects

## Quickstart

```sh
# download template
git clone https://github.com/alikhil/ts-node-template
# install dependencies
npm install

# run tests
npm test

# run gulp tasks
gulp
```

## Template

1. Implement `db.connect` function in src/db.ts
2. Configure middlewares and routes in app.ts
